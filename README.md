# ISU Leave System - Green Light Grey Full Version

## Features included
- centered green/light-grey login with slight transparency
- digital leave application page
- auto-calculation of inclusive leave days
- leave type dropdown: Vacation, Sick, Study, Others
- printable leave form and printable request history
- digital admin/head approval with remarks
- no signature section in the workflow
- updated database with position and leave form fields

## Setup
1. Copy the project folder into your XAMPP `htdocs` directory.
2. Import `database/isu_leave_system_modern.sql` in phpMyAdmin.
3. Optional: place your logo at `assets/logo.png` and background at `assets/login-bg.jpg`.
4. Open the project in your browser.
